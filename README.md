## Run Locally

Clone the project

```bash
  git clone https://github.com/MohitSaraff/Blog-Website.git
```

Go to the project directory

```bash
  cd Blog-Website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
